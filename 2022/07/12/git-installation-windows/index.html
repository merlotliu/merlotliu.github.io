Git 使用 SSH 连接，为了保证正确安全的连接，确认双方的身份，需要在本地创建 key，并添加到 Github 中。具体如下：

  1. 生成密钥（会生成在~/.ssh.id_rsa.pub，如果使用其他方式，将rsa的位置替换即可）：
  2. 打印密钥信息：
  3. 添加到 Github:
    在 Github 中，依次打开 settings —- SSH and GPG keys —- 新建。
    将密钥复制到下面的框中，标题随意。
  4. 检查是否成功，在git bash输入ssh -T git@github.com，出现用户名，即为成功。
